Spectrogram

constant-Q · note names · 24 bins/octave
How to read this →
Vertical axis: pitch as note names (C1 at bottom, C8 at top). Each note has two adjacent bins (24 bins/octave = quarter-tone resolution), so a bright band that straddles two bins between, say, C5 and C#5 means the played pitch is closer to C-quarter-sharp — i.e. actually sharp by ~25–50 cents. Dashed gray lines mark each C octave for orientation.

Horizontal axis: time in seconds (lesson wall-clock, not slice-relative).

Color: amplitude in dB relative to the loudest point in the window (magma colormap; bright yellow = loud, dark purple = silent). Vertical stacks of bright bands above a fundamental are harmonics — they're real, not separate notes.

Reading a chord: a single attack at time t usually shows multiple fundamentals lighting up simultaneously, each with its own harmonic stack.
